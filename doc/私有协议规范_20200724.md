# 私有协议规范
版本：V0.0.1
## 1.前言
1.文档约束范围，下位机（MCU）类的协议交互。
2.面向1对1，1对多
3.约束256个CMD的使用规范，

## 2.CMD规范
CMD 范围 **0~255**
* **0~127** 为共有命令，即特殊命令
* **127~255**为私有命令，可以自行分配
### 2.1 公有命令

**0~ 20**:配置相关
**30~ 50**:传输相关
**60~80**：

### 2.1.1 配置相关

|名称|define|cmd|说明|
|---|---|---|---|
|获取软件版本请求|CMD_VERSION_REQ|0||
|获取软件版本应答|CMD_VERSION_RESP|1||
|获取配置请求|CMD_GET_CONF_REQ|2||
|获取配置应答|CMD_GET_CONF_RESP|3||
|设置配置请求|CMD_SET_CONF_REQ|4||
|设置配置应答|CMD_SET_CONF_RESP|5||
|||||
|||||
